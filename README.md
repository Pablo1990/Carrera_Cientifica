# Carrera Científica

Videojuego web corto (en español e inglés) para adolescentes sobre decisiones reales en la carrera científica.

## Cómo jugar / How to play

1. Abre `index.html` en tu navegador (desde la carpeta del proyecto).  
   Open `index.html` in your browser (from the project folder).
2. Selecciona el idioma con los botones **ES** / **EN** en la esquina superior derecha.  
   Select the language with the **ES** / **EN** buttons in the top-right corner.
3. Pulsa **Comenzar** / Press **Start**.
4. Elige opciones en cada situación y observa el resultado del dado virtual.  
   Choose an option for each situation and observe the virtual die result.
5. Intenta avanzar lo suficiente para ganar el Premio Nobel.  
   Try to progress far enough to win the Nobel Prize.

## Qué enseña / What it teaches

- Elección y posible cambio de carrera universitaria.
- Doctorado, papers, precariedad y supervisión complicada.
- Migrar al Reino Unido (beneficios y dificultades).
- Conferencias internacionales, industria y descubrimientos inesperados.
- Pensamiento crítico con lenguaje cercano.
